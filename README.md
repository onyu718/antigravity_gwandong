#  관동별곡 인터랙티브 학습 플랫폼

> **지도를 활용하여 정철의 여정을 시각적으로 따라가는 고전시가 학습 플랫폼**
>  https://onyu718.github.io/antigravity_gwandong/

---

###  기획 의도
관동별곡은 내신과 수능을 불문하고 고등학생들이 국어 시간에 가장 어려워하고 기피하는 대표적인 고전시가입니다. 교과서 텍스트로만 작품을 읽다 보면 정철이 도대체 어디서 어디로 이동하고 있는 건지 지리적 거리감이 전혀 와닿지 않고, 여기에 생소한 고어까지 겹치다 보니 무작정 통암기만 반복하는 악순환이 생깁니다.

관동별곡의 핵심은 결국 정철의 발자취를 따라가는 '여정과 공간의 이동'에 있습니다. 그래서 고등학생들이 빽빽한 필기 위주의 암기에서 벗어나, 실제 지리적 위치를 눈으로 확인하며 입체적으로 공부할 수 있도록 카카오 지도 API를 연동한 시각적 플랫폼을 만들었습니다. 본문의 명승지와 지도를 매칭해 가며 작품의 흐름을 자연스럽게 습득하고, 총 17문항의 핵심 퀴즈를 통해 자신이 내신이나 수능 포인트를 제대로 이해했는지 확인하며 주도적으로 학습할 수 있도록 돕고자 합니다.

###  주요 기능
* **인터랙티브 지도 연동:** 가사 속 명승지 단락을 클릭하면 해당 실제 지리적 위치로 지도가 부드럽게 이동합니다.
* **위치 기반 퀴즈 (17문항):** 각 장소마다 연계된 시험 필수 퀴즈를 풀며 본문 내용을 복습합니다.
* **실시간 학습 진도율:** 상단에 내 학습 완료율(%)이 실시간으로 표시되어 공부 성취도를 보여줍니다.
* **로컬스토리지 활용:** 별도의 회원가입이나 로그인 번거로움 없이도 브라우저 자체에 내 학습 진도가 안전하게 저장됩니다.

###  개발 환경
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **API:** Kakao Maps API
* **Deployment:** GitHub Pages
